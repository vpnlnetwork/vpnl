# Quick Deploy Guide - TL;DR

**⏱️ Total Time: ~30 minutes**

-----

## Prerequisites (5 min)

```bash
# 1. Get testnet ETH
# Visit: https://faucet.quicknode.com/arbitrum/sepolia

# 2. Get Etherscan API key
# Visit: https://docs.etherscan.io/

# 3. Get your wallet private key (MetaMask)
# Account Details → Export Private Key (remove 0x prefix)
```

-----

## Setup (2 min)

```bash
# Install
npm install

# Configure
cp .env.example .env
nano .env  # Add your PRIVATE_KEY and ETHERSCAN_API_KEY
```

**.env format:**

```bash
PRIVATE_KEY=abc123...  # NO 0x prefix!
ETHERSCAN_API_KEY=YOUR_KEY_HERE
```

-----

## Verify Setup (1 min)

```bash
npm run verify:setup
```

**Expected:** “✅ All checks passed!”

-----

## Deploy (10 min)

```bash
# Deploy contract
npm run deploy:sepolia

# Create test data
npm run populate:testdata
```

## Troubleshooting

**“Insufficient funds”**
→ Get more ETH from faucet

**“Only verifier can call”**
→ Use same wallet that deployed

**“ETHERSCAN_API_KEY invalid”**
→ Wait 5 min for key activation

**“Network error”**
→ Check internet connection
